# Emile Izere — Portfolio

A personal portfolio organized into three interactive "rooms": Computer Science, Economics, and Data Science. Each room showcases projects, readings, skills I know, and what I'm currently learning.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Structure

- **Home** — Overview with three clickable room cards
- **Rooms** — CS, Economics, Data Science — each with projects, readings, skills
- **About** — Education, awards, communities, life beyond code

## Editing Content

Edit `src/data/portfolio.json` or use the admin panel at `/admin` (password: `admin123`)

## Tech Stack

- Next.js 15
- Tailwind CSS
- Framer Motion
- TypeScript

## Deploy

Push to GitHub, then import to [Vercel](https://vercel.com) for instant deployment.
